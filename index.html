TypeScript ≈ JavaScript

Большая экосистема и комьюнити

+ =

// @ts-check

Создаем машину

function createCar(
  rightHand: boolean,
  model: string,
  year: number): Car {
  let car;
  /* business logic */
  return car;
}

TypeScript кушает экономит ваше время

  1. Объясняет ваш код
  2. Защищает вас от банальных ошибок
  3. Дает свободу рефакторинга

Unit тесты? А может сначала TypeScript?

Когда использовать

  • Нужно быстро и точно понимать код
  • Это важно
  • Это сложно
  • Нужно рефакторить

Не нужно если

  • Это временное решение
  • Прототип
  • Это простое и маленькое
  • Вы единственный разработчик

    }

TypeScript загрузчик для webpack

module.exports = {
  mode: "development",
  entry: "./app.ts",
  output: { filename: "bundle.js" },
  resolve: { extensions: [".ts", ".tsx", ".js"] },
