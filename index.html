Решение - Типизация проекта

«Если это выглядит как утка, плавает как утка и крякает как утка, то, вероятно, это утка (какая разница, что это на самом деле)»

Создаем машину

function createCar(
  rightHand: boolean,
  model: string,
  year: number): Car {
  let car;
  /* business logic */
  return car;
}

Unit тесты? А может сначала TypeScript?

Когда использовать

  • Нужно быстро и точно понимать код
  • Это важно
  • Это сложно
  • Нужно рефакторить

Не нужно если

  • Это временное решение
  • Прототип
  • Это простое и маленькое
  • Вы единственный разработчик

Каковы самые слабые места Vue?

Сейчас настройка TypeScript во Vue проекте очень тривиальна

нужно всего лишь...
  1. Настроить tsconfig.json
  2. Настроить ваш сборщик под импорт TS
  3. Настроить импорт однофайловых компонентов *.vue
  4. Добавить в зависимости для декораторов
    • vue-class-component
    • vue-property-decorator

А может проще сову нарисовать?
